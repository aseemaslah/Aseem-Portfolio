# Portfolio Website

This is a modern, high-performance portfolio website built with:
- **React** (Vite)
- **Tailwind CSS v4** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Features
- **Responsive Design**: Looks great on all devices.
- **Smooth Animations**: Powered by Framer Motion.
- **Dark Mode Aesthetic**: Inspired by high-end design portfolios.
- **Optimized Performance**: Fast loading with Vite.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser.

3.  **Build for Production**:
    ```bash
    npm run build
    ```
    The output will be in the `dist` folder.

## Customization
- Update your project data in `src/components/Projects.jsx`.
- Update your skills in `src/components/Skills.jsx`.
- Change colors/fonts in `src/index.css` (Tailwind v4).

## Deployment
You can deploy the `dist` folder to any static host (Vercel, Netlify, GitHub Pages).
To deploy to GitHub Pages, you may need to set the `base` in `vite.config.js`.

Enjoy your new portfolio!
